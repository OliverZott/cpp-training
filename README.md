## Run and Debug

- use CLI, e.g.:
  - `g++ TernaryOperator.cc -o main`
  - `./main`
- Use makefile:
  - `make build` and `make run`, afterwards `make clean`
- Cleanup remains:
  - Windows `.\cleanup.ps`
  - Linux `/home/comp/dev/cpp-training/cleanup.sh`
